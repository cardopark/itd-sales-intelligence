# WGSN Sales Intelligence MVP

## 목표

기존 기업고객, 해지 고객, 마케팅 리드풀, 리멤버 명함풀, 공개 비즈니스 뉴스를 연결해 세일즈 매니저가 매월 우선적으로 컨택해야 할 기업과 담당자를 추천하는 서비스입니다.

핵심 목적은 세 가지입니다.

- 구독 기업과 닮은 신규 타겟 기업 발굴
- 해지 고객의 사업 확장 신호를 감지해 재구독 타이밍 발견
- 내부 리드풀에 담당자 정보가 있는지 확인하고 맞춤 컨택 액션 추천

## 사용자

- 세일즈 매니저: 이번 달 연락할 기업과 담당자, 메시지 포인트를 확인
- 세일즈 리더: 산업군별 파이프라인, 재구독 기회, 액션 현황을 관리
- 마케팅 담당자: 캠페인 리드 중 세일즈 전환 가능성이 높은 계정을 확인

## 입력 데이터

### 내부 데이터

- 현재 구독 고객사: 기업명, 산업군, 국가, 매출 규모, 브랜드 포트폴리오, 사용 부서, 계약 금액, 계약 시작/종료일
- 해지 고객사: 해지일, 과거 이용 부서, 해지 사유, 과거 계약 금액, 담당 세일즈
- 리드풀: 기업명, 담당자명, 직함, 부서, 이메일, 전화번호, 수집 채널, 최근 반응
- 리멤버 명함풀: 기업명, 담당자명, 직함, 부서, 연락처, 관계자
- CRM 활동 기록: 마지막 컨택일, 미팅 여부, 제안 여부, 딜 상태

### 외부 데이터

- 온라인 뉴스
- 기업 PR/보도자료
- IR 자료와 실적 발표
- 채용 공고
- 신제품 출시 페이지
- 유통 입점/팝업/콜라보 소식
- SNS/커뮤니티 반응 지표

### 자동 수집 데이터

비즈니스 신호는 수동 업로드 대상이 아니라 월 1회 자동 탐색 대상입니다.

- 뉴스 검색 API: 기업명, 브랜드명, 대표 제품명, 산업 키워드 조합으로 최근 30~45일 뉴스 검색
- PR/보도자료: 기업 공식 뉴스룸, 브랜드 사이트, 유통사 입점 뉴스, 캠페인 페이지 확인
- 공시/재무 API: 상장사는 OpenDART 등 공식 공시 API로 재무제표와 사업보고서 확인
- 비상장사 보강: 투자 유치, 채용, 신제품, 유통 확대, 팝업, 해외 진출 등 공개 신호 중심 확인
- 수집 결과 저장: signals 자동 생성 테이블에 출처 URL, 요약, 신호 유형, 신뢰도, 세일즈 의미 저장

## 핵심 기능

### 1. 유사 기업 추천

현재 구독 고객사를 기준으로 비슷한 산업군, 브랜드 포지션, 시장 확장 단계, 제품 카테고리, 글로벌 진출 여부를 가진 기업을 추천합니다.

추천 기준 예시:

- 같은 산업군과 카테고리
- 유사 가격대 또는 소비자 타겟
- 신제품 출시 빈도
- 해외 진출 또는 신규 유통 채널 확대
- 디자인, 컬러, 소재, 소비자 트렌드 의존도가 높은 사업 구조
- R&D, 상품기획, 브랜딩 조직의 존재

출력:

- 추천 기업명
- 추천 이유
- 유사 고객사
- 예상 니즈
- 세일즈 적합도 점수
- 추천 담당 부서

### 2. 해지 고객 재구독 신호 감지

해지 고객사를 매월 모니터링해 긍정적인 사업 신호를 찾습니다.

감지 신호:

- 신제품 출시 후 긍정 반응
- 흑자전환
- 매출 연속 증가
- 영업이익 증가
- 신규 브랜드 론칭
- 해외 시장 진출
- 대형 유통 채널 입점
- 투자 유치
- R&D, 디자인, 상품기획 채용 증가

출력:

- 이번 달 긍정 소식
- 왜 WGSN 재구독 타이밍인지
- 추천 컨택 메시지
- 우선순위
- 담당 세일즈 매니저

### 2-1. 월간 자동 비즈니스 신호 수집

매월 1회 고객/해지/추천 후보 기업을 대상으로 다음 순서로 자동 업데이트합니다.

1. 기업명과 브랜드명 후보를 정규화합니다.
2. 최근 30~45일 뉴스, PR, 공시, IR 자료를 검색합니다.
3. 중복 기사와 단순 언급 기사를 제거합니다.
4. 긍정 비즈니스 신호만 분류합니다.
5. 상장사는 재무제표와 사업보고서에서 매출, 영업이익, 순이익, 연구개발비, 사업부문 설명을 확인합니다.
6. 전년 동기 또는 직전 기간 대비 개선 여부를 계산합니다.
7. WGSN 구매 가능성과 연결되는 문장으로 세일즈 인사이트를 생성합니다.

자동 분류 신호:

- 매출 성장
- 영업이익 개선
- 흑자전환
- 신제품 출시
- 신규 브랜드/라인 론칭
- 해외 진출
- 유통 채널 확대
- 투자 유치
- R&D/디자인/상품기획 채용 증가
- 브랜드 리뉴얼
- 콜라보/팝업/전시 성과

재무제표에서 확인할 지표:

- 매출액
- 영업이익
- 당기순이익
- 매출총이익률
- 영업이익률
- 연구개발비
- 판매관리비
- 재고자산
- 사업부문별 매출

사업보고서에서 확인할 문맥:

- 신규 사업
- 연구개발 활동
- 해외 시장
- 브랜드/제품 전략
- 설비 투자
- 주요 계약
- 리스크 요인

### 3. 담당자 컨택 정보 탐색

추천 기업 또는 해지 고객 중 내부 리드풀/리멤버 명함풀에 담당자가 있는지 확인합니다.

매칭 기준:

- 기업명 정규화
- 도메인 매칭
- 브랜드명/법인명 매칭
- 담당자 부서와 직함 매칭
- 최근 반응과 관계 강도

우선 담당자:

- 상품기획
- 브랜드 전략
- 디자인
- CMF
- R&D
- 마케팅
- 소비자 인사이트
- 트렌드 리서치
- 전략기획

출력:

- 연락 가능한 담당자
- 연락 채널
- 관계 출처
- 추천 액션
- 메시지 초안

### 4. 맞춤 세일즈 전략 추천

기업별로 현재 상황에 맞는 접근 전략을 자동 제안합니다.

전략 유형:

- 유사 고객 레퍼런스 기반 접근
- 신제품 성과 기반 접근
- 해외 진출 지원 접근
- R&D/상품기획 예산 타이밍 접근
- 재구독 회복 접근
- 마케팅 리드 후속 접근

## 점수 체계

총점 100점 기준으로 우선순위를 계산합니다.

- 기업 적합도 30점: 산업군, 제품 주기, 트렌드 의존도
- 성장 신호 25점: 매출, 이익, 출시, 투자, 채용
- 구매 가능성 20점: 예산 부서 존재, 과거 구매 이력, 유사 고객 패턴
- 컨택 가능성 15점: 담당자 연락처, 관계 강도, 최근 반응
- 타이밍 10점: 계약 갱신기, 신제품 출시, 조직 개편, 캠페인 시점

우선순위:

- 80점 이상: 이번 달 1순위 컨택
- 65~79점: 캠페인 또는 웜업 컨택
- 50~64점: 모니터링 유지
- 50점 미만: 보류

## 월간 운영 흐름

1. 매월 1일 내부 고객/리드 데이터를 업데이트합니다.
2. 해지 고객사와 추천 후보 기업의 공개 뉴스를 수집합니다.
3. 긍정 비즈니스 신호를 분류하고 점수를 계산합니다.
4. 리드풀과 리멤버 명함풀에서 담당자 연락처를 매칭합니다.
5. 세일즈 매니저별 추천 리스트와 메시지 초안을 생성합니다.
6. 세일즈 매니저와 리더에게 월간 추천 기업 리포트 이메일을 발송합니다.
7. 컨택 결과를 CRM에 기록하고 다음 달 추천 점수에 반영합니다.

## MVP 화면

첫 버전은 네 개 화면이면 충분합니다.

- 이번 달 추천: 우선 컨택 기업 리스트
- 유사 기업 발굴: 구독 고객 기반 추천 기업
- 재구독 기회: 해지 고객 긍정 소식
- 담당자 매칭: 리드풀/명함풀 연락 가능성

## 기술 구성

### 빠른 MVP

- 데이터 입력: CSV 또는 Google Sheets
- 뉴스 수집: Google News, Naver News, 기업 PR 페이지 RSS 또는 검색 API
- 분석: LLM 기반 뉴스 요약과 신호 분류
- 저장: Airtable, Google Sheets, 또는 간단한 데이터베이스
- 화면: 웹 대시보드
- 알림: 이메일 또는 Slack

### 확장 버전

- CRM 연동: Salesforce, HubSpot, Pipedrive 등
- 명함/리드풀 연동: 내부 DB, 리멤버 export 데이터
- 기업 데이터 보강: DART, OpenDART, Crunchbase, Similarweb, 채용 플랫폼
- 자동화: 월간 배치 작업과 세일즈 매니저별 리포트
- 보안: 고객 데이터 암호화, 접근 권한, 감사 로그

## 자동 수집 파이프라인

### 1. 뉴스/PR 수집

검색어 예시:

- `{기업명} 신제품 출시`
- `{기업명} 매출 증가`
- `{기업명} 흑자전환`
- `{기업명} 투자 유치`
- `{기업명} 해외 진출`
- `{브랜드명} 팝업`
- `{브랜드명} 입점`
- `{기업명} 상품기획 채용`
- `{기업명} R&D 채용`

추천 소스:

- 네이버 뉴스 검색 API
- Google Programmable Search JSON API
- 기업 공식 뉴스룸과 PR 페이지
- 산업 전문 매체

### 2. 공시/재무 수집

국내 상장사는 OpenDART API를 우선 사용합니다.

필수 수집:

- 회사 고유번호
- 공시 목록
- 사업보고서/반기보고서/분기보고서
- 단일회사 주요 재무계정
- 단일회사 전체 재무제표
- 주요 재무지표

해외 상장사는 국가별 공시 API 또는 데이터 벤더를 추가합니다.

- 미국: SEC EDGAR
- 일본: EDINET
- 유료 보강: S&P Capital IQ, FactSet, PitchBook, Crunchbase, Similarweb 등

### 3. 신호 판정

자동 수집된 문서와 재무 데이터를 아래 기준으로 점수화합니다.

- 강한 긍정: 흑자전환, 영업이익 급증, 대형 투자 유치, 해외 시장 본격 진출
- 중간 긍정: 신제품 출시, 유통 확대, 신규 라인 론칭, 브랜드 리뉴얼
- 약한 긍정: 캠페인 진행, 단발성 팝업, 일반 채용
- 제외: 단순 광고성 기사, 부정 이슈, 확인되지 않은 루머, 관련성 낮은 기사

### 4. 세일즈 기회 변환

각 신호는 세일즈 액션으로 번역합니다.

- 신제품 출시: 컬러/소재/소비자 반응 리포트 기반 미팅 제안
- 매출 증가: 다음 시즌 확장 전략과 트렌드 리서치 예산 논의
- 흑자전환: 재구독 또는 소규모 컨설팅으로 재진입 제안
- 해외 진출: 지역별 소비자/리테일 트렌드 리포트 제안
- R&D 채용 증가: 연구개발 예산 확대 가능성으로 담당 부서 탐색

## 운영 자동화

권장 실행 주기:

- 매월 1일 오전: 전월 뉴스/공시/PR 수집
- 매월 1~2일: 자동 요약과 점수 계산
- 매월 첫째 주 세일즈 회의 전: 세일즈 매니저별 추천 리스트 배포

자동화 결과물:

- 이번 달 신규 긍정 신호
- 해지 고객 재구독 추천
- 신규 유사 기업 중 성장 신호가 있는 계정
- 담당자 연락처 매칭 결과
- 세일즈 메시지 초안
- 확인이 필요한 애매한 기사 목록

## 월간 이메일 알림

매월 자동 조회가 끝나면 세일즈 담당자별로 개인화된 이메일 리포트를 발송합니다.

수신자 유형:

- 세일즈 매니저: 본인이 담당해야 할 추천 기업과 액션만 수신
- 세일즈 리더: 전체 추천 기업, 담당자별 액션, 우선순위 요약 수신
- 마케팅 담당자: 마케팅 리드풀에서 컨택 가능한 기업과 캠페인 후속 액션 수신

이메일 내용:

- 이번 달 우선 컨택 기업
- 추천 이유
- 감지된 긍정 비즈니스 신호
- 담당자 연락처 매칭 여부
- 추천 연락 채널
- 메시지 초안
- 근거 뉴스/공시/PR 링크
- CRM에 기록할 다음 액션

발송 조건:

- 80점 이상 우선 컨택 기업은 담당 세일즈에게 자동 발송
- 해지 고객 중 강한 긍정 신호가 있으면 담당 세일즈와 세일즈 리더에게 발송
- 담당자 연락처가 없으면 리더에게는 요약만 보내고, 담당자 탐색 태스크를 생성
- 부정확하거나 애매한 신호는 "검토 필요" 섹션으로 분리

연동 방식:

- 빠른 MVP: SMTP, Gmail, Outlook 중 하나로 발송
- 확장 버전: CRM 태스크 생성과 이메일 발송을 함께 처리
- 보안: 수신자 목록, 고객 정보, 연락처는 내부 권한 정책에 따라 관리

필요한 설정값:

- NAVER_CLIENT_ID
- NAVER_CLIENT_SECRET
- GOOGLE_SEARCH_API_KEY
- GOOGLE_SEARCH_ENGINE_ID
- OPENDART_API_KEY
- 검색 대상 국가와 언어
- 제외 키워드
- 우선 산업군

## 예시 추천 결과

### 신규 타겟

- 기업: A 뷰티 브랜드
- 추천 이유: 기존 구독 고객인 프리미엄 스킨케어 브랜드와 유사한 제품 출시 주기, 해외 유통 확대, 브랜드 리뉴얼 진행
- 추천 부서: 상품기획, 브랜드 전략, 소비자 인사이트
- 액션: 기존 뷰티 고객 레퍼런스와 2027 컬러/패키징 트렌드 샘플 리포트로 미팅 제안

### 재구독 기회

- 기업: B 패션 기업
- 긍정 신호: 2개 분기 연속 매출 증가, 신규 라인 론칭, 해외 팝업 운영
- 추천 이유: 상품기획과 글로벌 시장 진출 단계에서 트렌드 리서치 니즈가 재발생할 가능성 높음
- 액션: 과거 사용 부서 담당자 또는 현재 브랜드 전략 담당자에게 재도입 미팅 제안
