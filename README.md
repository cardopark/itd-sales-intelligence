# WGSN Sales Intelligence

고가 B2B 구독/컨설팅 서비스의 신규 리드 발굴과 해지 고객 Win-back 타이밍을 자동으로 찾아주는 세일즈 인텔리전스 MVP입니다.

## 1. 문제 정의

WGSN은 컬러, 디자인, 소재, 소비자, 마케팅 트렌드를 분석해 기업에 제공하는 고가 구독/컨설팅 서비스입니다.  
서비스 특성상 구매 가능성이 높은 고객군이 넓지 않고, 단순히 많은 리드를 모으는 것보다 **실제로 구매 가능성이 있는 기업을 선별해 관리하는 것**이 중요합니다.

현재는 미팅이 잡히면 딜 전환율은 나쁘지 않지만, 아래 두 가지가 병목입니다.

- 기존 고객과 유사한 신규 타겟 기업을 체계적으로 발굴하기 어렵다.
- 해지 고객이 다시 구매를 검토할 만한 사업 확장 타이밍을 놓치기 쉽다.

## 2. 솔루션

기존 고객, 해지 고객, 보유 리드풀 데이터를 기반으로 세일즈 우선순위를 자동 추천하는 대시보드를 설계했습니다.

1. 기존 고객 리스트, 해지 고객 리스트, 리드풀/명함풀 데이터를 CSV로 업로드합니다.
2. 기존 고객과 유사한 산업군, 브랜드 포지션, 사업 확장 단계의 신규 타겟 기업을 추천합니다.
3. 해지 고객의 뉴스, PR, 공시, 재무자료를 월 1회 자동 확인해 긍정적인 비즈니스 신호를 탐지합니다.
4. 추천 기업의 관련 담당자가 내부 리드풀에 있는지 확인합니다.
5. 세일즈 매니저별로 이번 달 추천 기업, 담당자, 추천 액션을 이메일 리포트로 전달합니다.

## 3. 핵심 기능

- 기존 고객 기반 유사 기업 추천
- 해지 고객의 Win-back 기회 탐지
- 뉴스, PR, OpenDART 등 외부 데이터 기반 월간 모니터링
- 매출 증가, 흑자전환, 신제품 출시, 해외 진출, 투자 유치 등 긍정 신호 분류
- 리드풀/리멤버 명함풀 담당자 매칭
- 기업별 맞춤 세일즈 액션 추천
- 세일즈 매니저별 월간 이메일 리포트 발송

## 5. 사용 데이터

### 사용자가 업로드하는 데이터

- 기존 고객 CSV
- 해지 고객 CSV
- 마케팅 리드풀 CSV
- 리멤버 명함풀 또는 담당자 리스트 CSV

### 자동으로 수집/분석하는 데이터

- 웹 뉴스
- 기업 PR/보도자료
- 공시 자료
- 재무제표
- 사업보고서
- 채용/신제품/유통 확대 관련 공개 정보

## 6. 기대 효과

- 신규 리드 발굴을 “많이 찾기”가 아니라 “살 가능성이 높은 기업 찾기”로 전환
- 해지 고객의 재구독 타이밍을 놓치지 않고 포착
- 세일즈 매니저가 이번 달 연락할 기업과 담당자를 빠르게 확인
- 리드풀에 이미 있는 담당자를 활용해 컨택 성공률 향상
- 월간 리서치와 리포트 작성 업무 자동화
- 세일즈 파이프라인 확장과 Win-back 기회 극대화

## 7. 확장 가능성

- Salesforce, HubSpot 등 CRM 연동
- Gmail, Outlook 기반 이메일 자동 발송
- OpenDART, SEC EDGAR 등 재무/공시 API 확장
- 산업군별 추천 점수 모델 고도화
- 세일즈 액션 결과를 반영한 추천 정확도 개선
